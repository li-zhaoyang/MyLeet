344. Reverse String

Acceptances: 59.81 % Difficulty: Easy

Question

Write a function that takes a string as input and returns the string reversed.

Example:

  • Input: "hello" Output: "olleh".
  • Input: "qwerty1234" Output: "4321ytrewq".

Solution 1

Code Language: Java 

public class Solution {
    public String reverseString(String s) {
        char[] word = s.toCharArray();
        int i = 0;
        int j = s.length() - 1;
        while (i < j) {
            char temp = word[i];
            word[i] = word[j];
            word[j] = temp;
            i++;
            j--;
        }
        return new String(word);
    }
}

Complexity Analysis

Time Complexity: O(n)O(n) (Average Case) and O(n)O(n) (Worst Case) where nn is the total number character in the input string. The algorithm need to reverse the whole string.

Auxiliary Space: O(n)O(n) space is used where nn is the total number character in the input string. Space is needed to transform string to character array.

Algorithm

Approach: Iterative Swapping Using Two Pointers

One pointer is pointing at the start of the string while the other pointer is pointing at the end of the string. Both pointers will keep swapping its element and travel towards each other. The algorithm basically simulating rotation of a string with respect to its midpoint.

Solution 6

Code Language: Java 

public class Solution {
    public String reverseString(String s) {
        int length = s.length();
        if (length <= 1) return s;
        String leftStr = s.substring(0, length / 2);
        String rightStr = s.substring(length / 2, length);
        return reverseString(rightStr) + reverseString(leftStr);
    }
}

Complexity Analysis

Time Complexity: O(nlog(n))O(nlog(n)) (Average Case) and O(nlog(n))O(nlog(n)) (Worst Case) where nn is the total number character in the input string. The recurrence equation is T(n)=2T(n2)+O(n)T(n)=2T(n2)+O(n). O(n)O(n) is due to the fact that concatenation function takes linear time. The recurrence equation can be solved to get O(nlog(n))O(nlog(n)).

Auxiliary Space: O(h)O(h) space is used where hh is the depth of recursion tree generated which is log(n)log(n). Space is needed for activation stack during recursion calls.

Algorithm

Approach: Divide and Conquer (Recursive)

The string is split into half. Each substring will be further divided. This process continues until the string can no longer be divided (length 11). The conquering process will take they previously split strings and concatenate them in reverse order.
